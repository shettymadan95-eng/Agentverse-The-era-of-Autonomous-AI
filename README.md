# Agentverse: The Era of Autonomous AI

An intelligent AI agent that helps break down and execute complex tasks.

## Features
- Task planning with GPT-4
- Automated execution
- Memory storage
- Interactive web interface

## Tech Stack
- Node.js
- Express
- OpenAI API
- Bootstrap

## Setup
1. Clone the repository
2. Run `npm install`
3. Create `.env` file with OpenAI API key
4. Start with `npm start`
